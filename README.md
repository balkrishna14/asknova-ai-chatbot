# AskNova – AI Chatbot 🤖

AskNova is a modern AI-powered chatbot that allows users to interact with a large language model through a clean and responsive chat interface.  
The application is built using React, Tailwind CSS, Node.js, and Groq LLM APIs.

It demonstrates full-stack development including frontend UI, backend API integration, and cloud deployment.

# ⚙️ Installation

Clone the repository

git clone https://github.com/YOUR_GITHUB_USERNAME/asknova-ai-chatbot.git

Navigate to project

cd asknova-ai-chatbot

---

# ▶️ Run Backend

cd backend

Install dependencies

npm install

Run server

node server.js

Server runs on

http://localhost:3000

---

# ▶️ Run Frontend

Open another terminal

cd frontend

Install dependencies

npm install

Run development server

npm run dev

Open in browser

http://localhost:5173

---

# 🌐 Deployment

Frontend deployed using GitHub Pages.

npm run deploy

Backend deployed using Render.
